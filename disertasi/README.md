## Ringkasan
Penelitian disertasi ini bertujuan mengembangkan sistem prediksi gempa bumi yang komprehensif berbasis prekursor geomagnetik, dengan mengintegrasikan data multi-parameter melalui arsitektur EfficientNet-B0 hierarkis. Sistem dirancang dengan kemampuan self-updating dan online learning untuk adaptasi terhadap data baru secara kontinu.

**Kebaharuan**:
- Sistem prediksi komprehensif (bukan hanya deteksi)
- Prediksi parameter gempa: magnitudo, lokasi, waktu
- Integrasi multi-parameter (geomagnetik, seismik, ionosfer)
- Online learning untuk adaptasi real-time
